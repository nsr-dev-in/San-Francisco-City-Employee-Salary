# 📊 San Francisco Employee Salary Analysis

## 🚀 Project Overview

This project focuses on analyzing the **San Francisco City Employee Salary Dataset** using **Python Pandas**. The objective is to perform data cleaning, exploratory data analysis (EDA), and answer real-world business questions to extract meaningful insights from employee salary data.

Through this project, I solved **21 analytical questions** covering salary trends, employee information, job titles, and compensation analysis.

---

## 🎯 Objectives

* Understand the structure of a real-world dataset
* Perform data cleaning and preprocessing
* Handle missing values effectively
* Explore employee salary patterns
* Analyze job titles and departments
* Generate actionable insights using Pandas

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Jupyter Notebook

## 📂 Dataset

**Dataset:** San Francisco City Employee Salary Dataset

The dataset contains information about employee salaries, job titles, departments, benefits, and compensation details.
